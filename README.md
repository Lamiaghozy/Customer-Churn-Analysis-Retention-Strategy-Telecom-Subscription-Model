# Customer-Churn-Analysis-Retention-Strategy-Telecom-Subscription-Model

Executive Summary

This project evaluates customer churn within a subscription-based telecom model to diagnose revenue leakage and identify retention levers. Rather than reporting churn rates, the analysis isolates structural churn concentration across contract types, demographic segments, and usage behaviours.
By segmenting churn risk and modelling retention impact, the project highlights how targeted interventions can materially improve customer lifetime value (LTV) and recurring revenue stability.
Churn reduction is framed as a revenue protection strategy rather than a reporting metric.

Overview
This project analyses customer churn for a subscription-based telecom company (Databel). The objective was not only to calculate churn rates, but to diagnose the structural drivers of attrition and design data-driven retention strategies.
Using Excel-based modelling and dashboarding techniques, the analysis translates churn metrics into actionable commercial insights for revenue protection and lifetime value optimisation.
